レシピブック 017:aliasとundef

　Date：2013-12-20 19:53:30 +0900
　Categories： TECHNOLOGY

aliasの方です。

[5] pry(main)> def d1
[5] pry(main)* return "printing"
[5] pry(main)* end
=> nil
[6] pry(main)> alias d2 d1
=> nil
[7] pry(main)> d1
=> "printing"
[8] pry(main)> d2
=> "printing"

この状態でundefを使うます。

[9] pry(main)> undef d2
=> nil
[10] pry(main)> d2
NameError: undefined local variable or method `d2' for main:Object
from (pry):8:in ` __pry__'
[11] pry(main)> d1
=> "printing"

ちゃんとにエラーが出ます。
