git logの時々使うオプション

　Date：2018-02-28 02:24:22 +0900
　Categories： TECHNOLOGY

概要

git logの時々使うコマンドのメモ

コマンド

普通にコミットの履歴を見る

$ git log

一行でコミットの履歴を見る

$ git log --oneline

マージコミットを見る

$ git log --merges

マージコミット以外を見る

$ git log --no-merges

変更ファイルを見る

$ git log --name-status

変更行数を見る

$ git log --numstat

コミッターで絞る

$ git log --author='ryo'
