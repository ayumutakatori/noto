ファイルの開き方のあれこれと変数展開がテーマ（コードの整理がしたい。）

VBAからファイルを開く

Sub test()
Workbooks.Open "C: *********.csv"
End Sub

これは開けた。

変数に入れてから開く

Sub test()

Dim SalesCsvPass As String
SalesCsvPass = "C: *********.csv"

Workbooks.Open SalesCsvPass

End Sub

開いた。

最後（”C: ***************.csv”がひらきたい）

Sub test()

Dim WorkDir As String
WorkDir = "C: ******"

Dim SalesCsv As String
SalesCsv = " ******.csv"

Workbooks.Open WorkDir + "***" + SalesCsv
End Sub

これも開いた、ありがとう。ありがとう。
